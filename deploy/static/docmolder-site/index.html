Una decisione per volta.

DocMolder evita menu lunghi e passaggi inutili. Il bot pubblico guarda i file ricevuti e guida l'utente verso il prossimo passo sensato.

1

Manda file o foto

PDF, scansioni o immagini arrivano direttamente nella chat pubblica del bot.

2

Scegli l'azione

Il bot mostra solo opzioni compatibili: crea, comprimi, unisci, estrai, ruota.

3

Ricevi il risultato

Il documento torna pronto, con fallback conservativi quando servono.

Pensato per restare leggero.

DocMolder e pubblico su Telegram, ma resta leggero: la VPS esegue le trasformazioni, non diventa un archivio permanente di file personali.

  • File temporanei con retention breve.
  • Bot pubblico raggiungibile da @docmolder_bot.
  • Flusso principale via Telegram polling.
  • Nessun editor web o dashboard documentale.
  • Sito pubblico statico, senza proxy verso il runtime del bot.
