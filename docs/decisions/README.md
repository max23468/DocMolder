# Decisioni DocMolder

Questa cartella ospita decisioni autonome in formato ADR leggero.

Stato attuale:

- [../DECISIONS.md](../DECISIONS.md) resta il riepilogo storico delle decisioni
  stabili già prese.
- [../DECISIONS_PENDING.md](../DECISIONS_PENDING.md) resta il registro delle
  decisioni aperte.
- Le nuove decisioni strutturali dovrebbero essere aggiunte qui, usando
  [0000-template.md](./0000-template.md).

La migrazione delle decisioni storiche da `DECISIONS.md` va fatta
progressivamente, preservando contenuti, motivazioni e conseguenze operative.
