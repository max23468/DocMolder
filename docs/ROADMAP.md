## Focus 1.x

La linea 1.x non cambia il perimetro prodotto: `DocMolder` resta una utility
documentale Telegram-first, pubblica ma best-effort, con retention breve dei file
utente e senza storage documentale permanente.

Le priorita 1.x sono:

- mantenere affidabile il bot pubblico `@docmolder_bot` per uso reale a basso
  volume
- correggere regressioni e casi limite emersi dall'uso pubblico
- migliorare privacy, retention e cancellazione dati solo dentro il perimetro
  utility documentale
- rafforzare osservabilita, runbook e guardrail operativi prima di una
  promozione pubblica piu ampia

Restano fuori dal focus 1.x, salvo decisione esplicita:

- dashboard web-first
- archivio documentale permanente
- SLA pubblico o supporto commerciale strutturato
- API pubbliche o integrazioni esterne stabili
